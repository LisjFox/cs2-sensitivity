# 🎯 CS2 灵敏度小助手

> 为你寻找最适合的 CS2 鼠标灵敏度

一个基于 OpenClaw 平台的 AI 技能，能够自动识别玩家当前 CS2 配置、分析灵敏度设置、给出个性化调整建议，并支持一键写入游戏配置。

## 快速开始

1. 将本项目放入 OpenClaw 的 `skills/` 目录
2. 在 OpenClaw 中与 AI 对话：**"帮我调一下 CS2 灵敏度"**
3. AI 会自动读取你的 CS2 配置，引导你完成诊断与优化

## 功能

- 🔍 自动扫描本地 CS2 配置文件，读取灵敏度/分辨率
- 📊 计算 eDPI、cm/360° 等专业参数
- 🎯 基于角色定位（突破/狙击/自由人等）推荐最佳灵敏度
- 💾 一键写入配置，自动备份原文件
- 📈 （可选）通过 Steam API 获取战绩数据辅助分析

## 技术栈

- Python 3 — 核心计算与配置读写
- OpenClaw Skill Framework — AI 对话与工具编排
- Steam Web API — 战绩数据接口
